информационно-справочной системы общежития

Поиск

Поиск может осуществлять по Фамилии, Имени и Отчеству студента, введенных в порядке: Фамилия, Имя, Отчество. Пример: Иванов Иван Иванович

Поиск также может осуществлять по номеру студенческого билета

Фильтры

Для более комфортного поиска можно воспользоваться выпадающим списком фильтров по кнопке "Фильтры". Доступны фильтры: "Группа", "Комната", "Оплатил"

При выборе фильтра "Группа" появится дополнительный выпадающий список с доступными для фильтрации учебными группами студентов

При выборе фильтра "Комната" появится дополнительный выпадающий список с доступными для фильтрации комнатами, в которых сейчас проживают студенты

При выборе фильтра "Оплатил" появится дополнительный выпадающий список с вариантами "Да" и "Нет", позволяющий отфильровать студентов оплативших или не оплативших проживание в общежитии

Количество студентов на странице

Выпадающий список "Отображать по" дает возможность выбрать из предложенных вариантов, сколько студентов нужно отобразить на странице

Количество студентов на странице

Выпадающий список "Отображать по" дает возможность выбрать из предложенных вариантов, сколько студентов нужно отобразить на странице

Основная информация о студентах

В таблице отображается основная информация о студентах Фамилия, Имя, Отчество, Номер учебной группы, Номер студенческого билета, Номер комнаты в которой проживает студент и оплатил ли студент проживание в общежитии

Для более подробной информации о студенте необходимо зайти в редактирование карточки студента

Также с помощью кнопки в правом нижнем углу можно переключить режим отображения основной информации о студентах с таблицы на карточки с кратким описанием

Редактирование карточки студента

Если основаная информация о студенте отображается в виде таблице, то для редактирования карточки студента необходимо нажать кнопку находящуюся справа от информации о студенте

Если основаная информация о студенте отображается в виде карточек с кратким описанием, то для редактирования карточки студента необходимо нажать на саму карточку

При нажатии на редактирование карточки студента откроется модальное окно, содержащее заблокированные поля с информацией о студенте

Для того, чтобы отредактировать карточку студента необходимо нажать на кнопку "Редактировать", которая разблокирует для изменения поля с информацией о студенте

Нельзя оставлять поля с информацией о студенте пустыми

Нельзя вписывать в поле "Номер студенческого билета" номер, уже существующий в таблице

В поле "Номер комнаты" можно вводить только числа, номер комнаты не может содержать буквы

Важно!!! После внесения изменений в карточку студента нужно нажать кнопку "Сохранить" для сохранении внесенных изменений. Если кнопка "Сохранить" не будет нажата, внесенные изменения не сохранятся!

Добавление нового студента

Под всей информацией о студентах находится кнопка "Добавить студента"

При нажатии кнопки "Добавить студента" появится модальное окно добавления студента, содержащее поля в которые нужно внести всю информацию о студенте

Нельзя оставлять поля с информацией о студенте пустыми

Нельзя вписывать в поле "Номер студенческого билета" номер, уже существующий в таблице

В поле "Номер комнаты" можно вводить только числа, номер комнаты не может содержать буквы

Важно!!! После внесения информации в карточку нового студента нужно нажать кнопку "Сохранить" для сохранении информации. Если кнопка "Сохранить" не будет нажата, новый студент не будет создан!

Удаление студента

Если основаная информация о студенте отображается в виде таблице, то для удаления студента необходимо нажать кнопку находящуюся справа от информации о студенте

Если основаная информация о студенте отображается в виде карточек с кратким описанием, то для удаления студента необходимо нажать на в правом верхнем углу карточки

При нажатии на кнопку удаления студента откроется модальное окно удаления студента из таблицы, содержащее две кнопки "Да" и "Отмена"

При нажатии на кнопку "Да" студент будет удален из таблицы и модальное окно закроется

При нажатии на кнопку "Отмена" студент не будет удален из таблицы и модальное окно закроется

Изменение визуальной темы

При нажатии на кнопку можно изменить отображение визуальной темы информационной-справочной системой общежития вуза со светлой на темную и обратно.

Инструкция

Данная инструкция открывается при нажатии на кнопку , находящуюся в верхнем правом углу.
